Observação: todos os dados devem vir da base, não é para criar mock.

A universidade FIAP deseja criar um e-commerce para vender produtos personalizados. Para começarmos a primeira entrega, definimos algumas funcionalidades que já possam agregar valor.

Cadastros necessários (Criar e Listar além de inserção na base de dados)

  • Cadastrar fornecedores : Nome, endereço e telefone; (2 pontos)
  • Cadastrar clientes Nome, endereço e telefone; (1 ponto)
  • Categoria (Descrição e identificador); (1 ponto)
  • Produtos (Descrição, preço, fornecedor e categoria). (2 pontos)
  • Criar o Frontend (Instruções abaixo). (2 pontos)
  • Versionar o código no Git e criar um readme com o nome e RM dos integrantes. (1 ponto)
  • Criar um documento word documentando todos os testes realizados, bem como prints e comprovantes de funcionamento. (1 ponto)

Frontend – Tela de produtos

Ao cadastrar os produtos, teremos uma simulação de apresentação por categoria. Exemplo: caso o cliente selecione categoria feminina, mostrar apenas produtos cadastrados para esta categoria.
